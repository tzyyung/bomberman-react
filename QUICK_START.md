# 炸彈超人 React 版 - 快速啟動指南

## 🚀 快速開始

### 1. 安裝依賴
```bash
npm install
```

### 2. 啟動開發服務器
```bash
npm start
```

遊戲將在 http://localhost:3000 上運行！

### 3. 構建生產版本
```bash
npm run build
```

### 4. 部署
```bash
npm run deploy
```

## 🎮 遊戲控制

### 玩家 1 (藍色)
- **移動**: W A S D
- **炸彈**: 空白鍵
- **踢炸彈**: B 鍵
- **遙控**: V 鍵

### 玩家 2 (紅色)
- **移動**: 方向鍵
- **炸彈**: Enter 鍵
- **踢炸彈**: 右 Shift 鍵
- **遙控**: / 鍵

### 通用
- **暫停/繼續**: ESC 鍵

## 🎯 遊戲目標

- 使用炸彈摧毀對手
- 收集道具增強能力
- 避免被炸彈炸到
- 成為最後的倖存者！

## 🐛 常見問題

### 音頻無法播放
- 某些瀏覽器需要用戶交互才能播放音頻
- 點擊遊戲畫面後再嘗試

### 遊戲卡頓
- 確保使用現代瀏覽器
- 關閉其他標籤頁釋放內存

### 控制不響應
- 確保遊戲畫布有焦點
- 點擊遊戲畫面後再使用鍵盤
